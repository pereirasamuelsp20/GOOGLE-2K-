<h2>Under development</h2>
<p><h4>
ReliefMesh is a decentralized emergency response platform built for disaster scenarios where traditional internet infrastructure fails.<br>
Citizens can send one-tap SOS alerts, report road blockages, and navigate safely using a crowd-sourced adaptive map that fades stale <br>
data automatically.When cellular networks go down, ReliefMesh falls back to a peer-to-peer mesh network using Bluetooth Low Energy — devices<br>
relay messages hop- by-hop across nearby phones, ensuring SOS alerts still reach responders even without a single bar of signal.<br>
Built with React Native, Firebase, and Google Maps.<br>
